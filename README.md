# Project Overview

### 1. Name of your project?
**Keen Keeper**

### 2. A short description
Keen Keeper is a modern, user-friendly task management web application designed to help users organize their daily activities efficiently. The platform provides a clean interface for tracking goals and managing responsibilities, ensuring that nothing important slips through the cracks.

### 3. Technologies you used
* **Core Framework:** React.js
* **Styling:** Tailwind CSS 
* **Navigation:** React Router DOM
* **Notifications:** React Toastify
* **Icons:** React Icons
* **Deployment:** Netlify

### 4. 3 key features of your project
* **Interactive Dashboard:** A centralized hub where users can view and manage their tasks with a visually appealing and organized layout.
* **Responsive Design:** Fully optimized for all screen sizes, providing a seamless experience whether accessed via a desktop, tablet, or smartphone.
* **Dynamic Task Management:** Allows users to efficiently categorize and update their progress, helping them stay focused on their most important priorities.
